# Networking Basics Learned (Day 2)

- LAN: Local Area Network (e.g. home, office)  
- MAN: Metropolitan Area Network (city-level)  
- WAN: Wide Area Network (internet)  

## Common Port Numbers
- 21 → FTP  
- 22 → SSH  
- 25 → SMTP  
- 80 → HTTP  
- 443 → HTTPS  

## OSI Model (7 Layers)
1. Physical  
2. Data Link  
3. Network  
4. Transport  
5. Session  
6. Presentation  
7. Application
